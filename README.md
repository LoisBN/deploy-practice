# Your First Deploy — Practice App

A minimal React Router v7 app with the Vercel adapter pre-installed. Fork it, deploy to Vercel, see it live in under 2 minutes.

## Quick Start

**1. Fork this repo** — Click the **Fork** button at the top right of this page.

**2. Clone your fork:**
```bash
git clone https://github.com/YOUR-GITHUB-USERNAME/deploy-practice.git
cd deploy-practice
npm install
npm run dev
```

**3. Deploy to Vercel:**

Go to [vercel.com/new](https://vercel.com/new), find your forked repo and click **Import**, then click **Deploy** (no environment variables needed). Wait ~60 seconds — your app is live!
